## Introduction
```
1. 2020년 1월 11일 기준 청와대 국민청원 게시판 크롤링. 
2. 데이터 가공을 거쳐 데이터 시각화 구현.
3. 청원 게시판에서 가장 많이 업로드 된 텍스트 키워드 20개를 추출.
4. 빈도수가 가장 높은 키워드가 가장 큰 크기로 보여진다.
```
